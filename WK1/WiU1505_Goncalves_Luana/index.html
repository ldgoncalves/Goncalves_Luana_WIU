Project Description

With the Internet becoming more mobile first than desktop first, WorldSurf is one of the any applications that will be designed and tested for that concept. Furthermore, this project emphasizes the notion of designing for user intent to accomplish the principles of learnability, efficiency, memorability and satisfaction by building the application with consistency and familiar patterns. The landing page was created to list usability tests for mobile forms, interaction with the designs of the app and how it brings a pleasant navigating experience. Besides testing for user experience, the usable details page, usable homepage and categories pages, and mobile iterations will be tested as well. The final step would be migrating the design to desktop iterations. As each week of this month goes on, a new nav flow test will be listed below.
